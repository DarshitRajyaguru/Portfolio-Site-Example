FAQ

Frequently Asked Questions

Everything you need to know before we start working together.

How long does a typical WordPress project take?
+
Project timelines depend on scope and complexity. A standard 5–10 page website typically takes 3–4 weeks. WooCommerce stores range from 4–8 weeks. Custom plugin development varies from 1–6 weeks. I'll provide a detailed timeline in your project proposal.
Do you work with existing WordPress sites or only new builds?
+
Both! I work on new WordPress builds from scratch and also handle redesigns, migration, troubleshooting, and enhancements of existing WordPress sites. Whether it's a full rebuild or just adding specific features, I've got you covered.
What's included in your maintenance plans?
+
My maintenance plans include weekly updates (WordPress core, themes, plugins), daily automated backups with offsite storage, security monitoring and malware scanning, uptime monitoring, monthly performance reports, and a set number of development hours for minor changes and support.
Will I be able to edit my site after it's built?
+
Absolutely! I build all sites with intuitive admin interfaces and provide training so you can easily update content, images, blog posts, and products yourself. I typically use Elementor or the Gutenberg block editor for easy content management — no coding required.
Do you provide hosting for WordPress sites?
+
I don't provide hosting directly, but I partner with premium managed WordPress hosts like WP Engine, Kinsta, and SiteGround. I can help you choose the right hosting plan for your needs and handle the complete server setup and deployment.
What payment methods do you accept?
+
I accept bank transfers, PayPal, Stripe (credit/debit cards), and cryptocurrency (Bitcoin/USDC). Projects are typically structured with a 50% deposit upfront and 50% upon project completion. Monthly retainers are billed at the start of each month.
